# 방법론 노트

## 1. 프로젝트 목적

포트폴리오 수익률 표본에서 Historical VaR와 Expected Shortfall을 추정하고 예외 발생률을 확인한다.

## 2. 핵심 질문

과거 손익 분포로 추정한 VaR는 실제 손실 예외를 어느 정도 설명하는가?

## 3. 입력 데이터

- 파일: `data/sample/portfolio_returns.csv`
- 성격: 구조 검증용 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 수익률 하위 분위수로 Historical VaR를 계산하고 더 나쁜 손실 평균을 ES로 둔다.
2. 예외 횟수와 예외율을 같이 보여 모델 검증의 첫 단계를 만든다.
3. 표본은 손익 분포와 예외율 계산을 설명하도록 구성한 합성 수익률이다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
